# 项目名称：iOS文件阅读编辑器
- 简介：用于对纯文本文件进行编辑、预览，并且对部分常见的非纯文本进行阅读

## 项目简单介绍
### 1.功能
- txt、md、html文件编辑及预览
- 图片、docx、pdf文件浏览
- md转换html，html转换pdf

### 2.项目定位
- 类型：客户端、前端应用
- 平台：iOS
- 性质：个人开发项目

### 3.使用技术栈
- iOS、UIKit
- OC、Swift
- 第三方库：Down(Markdown文件相关库)


### 4.项目使用
- 导入xcode，连接iOS，运行并信任
- 使用方法：
    打开App
    或者通过其他app，选取文件，点击其他应用打开，选择本app，即可阅读文件

### 5.核心设计说明
- 本项目采用MVC，降低耦合性，方便后续维护
